// 설치순서 

1. VirtualBox 실행

2. VirtualBox 새로만들기 클릭
	-> 폴더명 : UbuntuTestServer
	-> 머신폴더 -> 기타 -> C:\Users\wante\VirtualBox\LinuxServer 
	-> 종류 : Linux
	-> 버전 : Ubuntu (64-bit)
	-> 다음 클릭

3. 메모리크기 
	-> 2048 MB
	-> 다음 클릭

4. 가상 머신 만들기
	-> 지금 새 가상 하드 디스크 만들기 -> 선택
	-> 만들기

5. 하드 디스크 파일 종류
	-> VDI (VirtualBox Disk Image) -> 선택
	-> 다음

6. 물리적 하드 드라이브에 저장
	-> 동적할당 -> 선택
	-> 다음

7. 파일 위치 및 크기
	-> 파일위치 : 변동없음
	-> 하드디스크 크기 : 50.00 GB
	-> 만들기

8. VirtualBox 설정 클릭 
	-> 저장소 클릭
		-> 컨트롤러: IDE - 비어있음 클릭
		-> 광학 드라이브 CD모양 클릭
		-> choose a dis file... 클릭
  		-> 위에서 다운받은 ubuntu 디스크파일 선택 
	-> 확인 누르고 설정 종료

9. VirtualBox 시작 클릭 
	-> 시동디스크 선택 -> 시작 클릭

10. Ubuntu 설치 과정
	-> Welcome
		-> 한국어 선택 
		-> Ubuntu 설치 클릭
	-> 키보드 레이아웃
		-> 계속하기 클릭
	-> 업데이트및 기타 소프트웨어
		-> 일반설치 선택
		-> 기타설정 -> 그래픽과 Wi-Fi 하드웨어... 체크
		-> 계속하기 클릭 
	-> 설치형식
		-> 디스크를 지우고 Ubuntu 설치 선택
		-> 지금 설치 클릭 -> 계속하기 클릭
	-> 어디에 거주하고 계십니까?
		-> 서울 선택 -> 계속하기 클릭
	-> 당신은 누구십니까?
		-> 이름 : UbuntuTest
		-> 암호 : 입력 -> 계속하기 클릭
	-> 설치화면
		-> 설치 상태 바 위에 Skip 버튼 클릭
		-> 설치완료 -> 지금 다시 시작 클릭

11. 까만 화면 -> enter 

12. 로그인 -> 우분투 바탕화면 -> 설치완료
